## Nomination Pools

<ul className="flex_Col2">
  <SettingItem
    title={<span>Claim Rewards</span>}
    description={
      <span>
        Claims all pending staking rewards from a nomination pool. This calls{' '}
        <code>nominationPools.claimPayout</code> to transfer any unclaimed
        rewards directly to the nominator&apos;s account.
      </span>
    }
  />
  <SettingItem
    title={<span>Compound Rewards</span>}
    description={
      <span>
        Bonds any unclaimed nomination pool rewards back into the pool,
        increasing the nominator&apos;s stake. This calls{' '}
        <code>nominationPools.bondExtra</code>, which automatically claims and
        bonds pending rewards in a single step.
      </span>
    }
  />
</ul>
